# The REAL Rating Form

This Streamlit application allows users to complete a multi-category survey based on the REAL Rating Form. The app displays one question at a time, collects user responses, and moves through different categories of questions.

## Features

- Displays one question at a time
- Supports multiple categories of questions
- Tracks user responses
- Allows saving responses to a CSV file

## Prerequisites

- Python 3.7 or higher
- Streamlit
- Pandas

## Getting Started

```bash
    pip install streamlit pandas
```
```bash
    streamlit run real_rating_form.py
```
